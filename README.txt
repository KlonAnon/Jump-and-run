Informatikprojekt - ein simples jump and run game:

Ziel ist es ein Spiel zu programmieren. Es gibt einen Spieler, welcher vor und zurück laufen und springen kann.
(eventuell weitere Optionen: Ducken, sprinten, interagieren mit Wänden oder anderen Objekten)
Es soll einfache (verschiedene) Gegner sowie (verschiedene) Hindernisse geben. Auch andere Extras, wie Blöcke, Münzen etc. sind denkbar.
Eben ganz im simplen Mario-Style
Weitere Features: Sound, Bilder(selbstverständlich), Game Menu

Vorgehensweise:
1.Erstellung des main game loops + fenster zur anzeige ======== erledigt
2.event handler + viereck, das sich bewegen und springen kann ====== erledigt
3.spieler soll nicht aus Window laufen können ====== erledigt
4.erstellen einer enemy class, die es ermöglicht mehrere feinde zu zeichnen (erst noch als viereck)
5.einfache interaktion zwischen spieler und feind
--> ähnlich zu Google T-Rex Game
6.hinzufügen eines hintergrundbildes, das mit bewegung des spielers interagiert
8.hinzufügen von boden der sich bewegt
9.hinzufügen von bildern für spieler und feind + animationen
...
